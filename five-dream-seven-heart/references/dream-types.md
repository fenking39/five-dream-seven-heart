## 一、六梦体系（按原因分类）

根据《周礼·春官·占梦》，占梦官以日月星辰占"六梦"之吉凶。这是传统解梦理论中最早、最权威的分类体系。

### 1. 正梦（正梦）

**定义**：无所感动，平安自梦。心无杂念、不受外界刺激时做的自然之梦，不带明显的悲喜色彩，是最"纯净"的梦。

**特征**：
- 梦境平缓、自然
- 无强烈情绪波动
- 往往是日常场景的温和再现

**解读**：正梦多代表内心平静，预示运势平稳。如梦中有象征性意象，可作为参考，但吉凶倾向不明显。

### 2. 噩梦（噩梦）

**定义**：惊愕而梦。因受到惊吓，在睡眠中延续恐惧情绪所做的梦。

**特征**：
- 伴随强烈的恐惧、焦虑情绪
- 可能梦到危险、威胁、追逐
- 醒后可能有余悸

**解读**：
- 心理学：揭示被压抑的焦虑、恐惧或内心冲突。噩梦是潜意识"敲门"的信号，解析往往带来突破。
- 玄学：需要警惕梦中的警示。噩梦多属"惧梦"，但也可能是"反梦"，阴极则吉，预示转折。需结合具体情境判断。

### 3. 思梦

**定义**：觉时所思念之而梦。即"日有所思，夜有所梦"。思念某人或沉溺于某事，因意念执着而入梦。

**特征**：
- 梦境内容与白天思考、牵挂的事物直接相关
- 常梦到渴望见到的人、渴望得到的事物
- 情感色彩明显（思念、期待、焦虑）

**解读**：
- 心理学：反映内心真实的渴望、牵挂或未解的情结。是了解自己真实需求的窗口。
- 玄学：思梦多与"意念"相关，若意念纯善，则为吉兆；若执念过深，可能导致运势偏差。建议调节心态，勿过度执着。

### 4. 寤梦

**定义**：觉时所道之而梦。因醒时的言行、听说所致，是白天的见闻在睡梦中的复现。

**特征**：
- 梦境场景是白天经历过的地点、事件
- 梦中人物是白天见过的人
- 梦境内容较为写实，象征性较弱

**解读**：
- 心理学：大脑整理白天信息的正常过程，象征意义相对有限。可关注梦中重复出现的细节，可能暗示潜意识关注点。
- 玄学：寤梦多为"复现"，吉凶意义较弱。若梦境中某个细节让你印象深刻，可重点关注，可能是"外应"。

### 5. 喜梦

**定义**：喜悦而梦。因心情愉悦、兴奋过头而做的美梦，即民间所谓的"笑醒"之梦。

**特征**：
- 情绪愉悦、满足
- 梦境内容通常是愿望达成的场景
- 醒后心情舒畅

**解读**：
- 心理学：反映内心深处的快乐、满足或愿望实现的心理预演。是积极情绪的体现。
- 玄学：喜梦多属"吉兆"，但需警惕"阳极则凶"。如果梦境过于完美、不真实，可能暗示虚幻，现实中需要保持清醒。适度喜悦为吉，过度兴奋则需谨慎。

### 6. 惧梦

**定义**：恐惧而梦。因心怀恐惧而入梦。

**特征**：
- 伴随恐惧、紧张情绪
- 梦境内容与内心的恐惧直接相关
- 可能是重复性的恐惧梦

**解读**：
- 心理学：揭示内心深处的恐惧、不安或创伤。需要正视恐惧的根源，寻求解决之道。
- 玄学：惧梦属"凶兆"较多，但也可能提示需要警惕的风险。若梦到恐惧场景但最终化解，则为"转危为安"之兆。建议加强自身修养，提升应对风险的能力。

## 二、梦型判定方法

### 1. 通过起因判定

**问题引导**：
- "做梦前一天，你睡前在想什么？" → 若明确想到某事，可能为思梦
- "睡前有受到惊吓或看恐怖片吗？" → 若有，可能为噩梦
- "睡前心情如何？很兴奋吗？" → 若兴奋，可能为喜梦
- "睡前有担心什么吗？" → 若担心，可能为惧梦

### 2. 通过梦境特征判定

| 梦型 | 情绪基调 | 内容特点 | 象征性 |
|------|---------|---------|--------|
| 正梦 | 平和 | 自然、日常 | 中等 |
| 噩梦 | 恐惧 | 危险、威胁 | 强 |
| 思梦 | 思念/牵挂 | 与白天思考相关 | 中等 |
| 寤梦 | 平淡 | 白天见闻复现 | 弱 |
| 喜梦 | 喜悦/满足 | 愿望达成 | 中等 |
| 惧梦 | 恐惧/紧张 | 恐惧场景复现 | 强 |

### 3. 复合梦型

一个梦可能同时具备多个梦型的特征。例如：
- 梦到白天考试（寤梦）但因担心没考好而紧张（惧梦） → 寤梦+惧梦
- 想念远方亲人（思梦）后梦见与之团聚（喜梦） → 思梦+喜梦

**判定原则**：以主导情绪和起因为主要判断依据。

## 三、梦日溯源

梦日即做梦前一天（或当天白天）的经历，是构建梦的"建筑材料"。追溯梦日有助于理解梦的素材来源。

### 1. 梦日刺激源类型

- **视觉刺激**：看到的广告、画面、场景
- **听觉刺激**：听到的话语、音乐、新闻
- **人际互动**：对话、交流、冲突
- **心理活动**：思考、担忧、期待
- **身体感觉**：疲劳、饥饿、不适

### 2. 梦日溯源方法

**问题引导**：
- "做梦前一天，你印象最深的事情是什么？"
- "那天有没有看到什么特别的画面或听到什么特别的话？"
- "那天和谁有互动？让你印象深刻吗？"
- "那天身体感觉如何？累不累？有没有不适？"

### 3. 梦日与梦境的关联

梦日元素本身不重要，但它与潜意识欲望结合后，产生象征意义。

**示例**：
- 梦日：看到一个"路牌"
- 潜意识欲望：渴望"方向指引"
- 梦境：梦到"迷路后看到老人指路"
- 关联：路牌（梦日素材）+ 指引（潜意识欲望）= 老人指路（梦境象征）

### 4. 躯体刺激溯源

睡眠中的身体感觉也可能成为梦的素材。

**常见躯体刺激与梦境对应**：

| 躯体刺激 | 可能的梦境 | 原理 |
|---------|-----------|------|
| 口渴 | 梦到喝水、找水 | 渴望被"借用"为欲望达成的媒介 |
| 憋尿 | 梦到找厕所、上厕所 | 同上 |
| 寒冷 | 梦到冰雪、冬景 | 身体感觉直接反映 |
| 疼痛 | 梦到受伤、被打 | 疼痛感转化为象征性场景 |
| 心跳加速 | 梦到奔跑、追逐 | 身体兴奋反映为梦中动作 |

**追溯方法**：
- "那天睡觉前身体感觉如何？累吗？饿吗？"
- "睡眠中有被吵醒或感觉不适吗？"

## 四、梦型与解析框架的对应

### 科学解析视角

- **正梦**：多为信息整理，象征意义有限，可简单分析
- **噩梦**：重点解析，揭示深层焦虑和冲突
- **思梦**：直接反映渴望和牵挂，是欲望分析的线索
- **寤梦**：信息复现，象征意义较弱，可略过
- **喜梦**：反映积极情绪和愿望达成，可分析满足的欲望
- **惧梦**：揭示恐惧根源，需要正视和处理

### 玄学解析视角

- **正梦**：运势平稳，无特殊吉凶
- **噩梦**：需警惕，可能为凶兆或反梦（阴极则吉）
- **思梦**：执念之梦，需看意念善恶
- **寤梦**：外应之梦，吉凶意义较弱
- **喜梦**：吉兆，但需警惕阳极则凶
- **惧梦**：凶兆，需谨慎防范

## 五、梦型在报告中的呈现

在"五梦七心相"报告中，梦型归类位于【一、五梦归类】部分：

```
【一、五梦归类】
├── 梦型判定
│   └── 正梦 / 噩梦 / 思梦 / 寤梦 / 喜梦 / 惧梦
└── 梦日溯源
    ├── 近期印象：[梦日关键事件]
    └── 躯体刺激：[睡眠身体感觉]
```

**呈现示例**：

【一、五梦归类】
- 梦型判定：思梦 + 惧梦（因思念亲人且担心其健康而入梦）
- 梦日溯源：
  - 近期印象：白天收到远方亲人消息，得知其身体不适
  - 躯体刺激：睡眠时感到寒冷，可能加强了"担心"的情绪

## 注意事项

1. 梦型判定不必过于精确，抓住主要特征即可
2. 复合梦型很常见，不要强行归为单一类型
3. 梦日溯源不需要面面俱到，关注最可能的关联即可
4. 躯体刺激的追溯需谨慎，不要过度解读
5. 梦型本身不是吉凶定论，需结合梦境内容和解析框架综合判断
