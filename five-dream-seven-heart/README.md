## 🌙 这是什么？

「五梦七心相」是一个基于 **TRAE SOLO** 的梦境解析技能。它不同于网上那些千篇一律的周公解梦词典——它像一个会"聊天"的朋友，通过多轮温和的对话，帮你探索这个梦对你而言的独特意义。

**双视角解读**：弗洛伊德心理学 × 周易传统易学，两种智慧体系对照互补。

**双模式设计**：深度模式（11-18轮）适合深度探索，快速模式（3-5轮）适合快速了解。

## 🎯 适合谁用

- 做了一个奇怪的梦，想知道意味着什么
- 重复做同一个梦，想了解背后的心理模式
- 被噩梦困扰，想找到情绪的根源
- 对自己的潜意识好奇，想通过梦境探索内心
- 想从传统视角了解梦境的"预示"意义

> 不需要心理学背景，不需要玄学知识，你只需要带着一个梦来。

## 🚀 如何使用

### 在 TRAE SOLO 中安装

1. 打开 TRAE SOLO 桌面端或网页端
2. 进入 **技能管理** 页面
3. 点击 **导入技能**，选择本仓库中的 `five-dream-seven-heart` 文件夹
4. 系统自动解析后即可使用

### 使用步骤

```
你：我昨晚梦见自己从高处坠落，然后飞了起来

五梦七心相：你带着一个梦来了，是吗？我感受到这是一个充满张力的梦——
         坠落时的不安和飞翔时的自由，两种感觉交织在一起。
         这种感觉，你还记得多少？

你：记得很清楚

五梦七心相：你想怎么探索这个梦？你可以选：
         A. 🔬 心理学视角 — 深度对话探索（11-18轮）
         B. 🏯 传统易学视角 — 深度对话探索（11-18轮）
         C. 🔮 两者都要 — 深度对话探索（11-18轮）
         D. ⚡ 快速解读 — 3轮快速了解

你：选C，两者都要

→ 然后AI会引导你描述更多的梦境细节
→ 11-14轮对话后，生成专属心相分析报告（HTML格式）
```

## 📚 参考来源

### 心理学框架
- 弗洛伊德《梦的解析》— 梦是欲望的伪装达成
- 朱建军《大众心理学·解梦500例》— 自由联想法与象征词典
- 梦境六步解析法：记录→寻源→联想→识象→重构→实践

### 传统易学框架
- 《周礼·春官·占梦》— 六梦体系（正梦、噩梦、思梦、寤梦、喜梦、惧梦）
- 《易经》八卦万物类象— 梦象转卦象法
- 五行生克与反梦法— "阴极则吉，阳极则凶"
- 五不占、五不验原则— 解梦的严谨态度
